# carmen
## Setup for PXC200AF device
```
1- yum install firefox
2- /usr/src/kernels
yum install kernel-devel (skip this step if you are not using FXVD4)
3- run updates
4-edit sudoers file
5-edit /etc/yum.repos.d/dropbox.repo
6-install dropbox (download files)
7- yum install gcc && yum instal gcc-c++
8-yum install make
9-yum install mod_wsgi
10- install Carmen For Linux (skip this step if you are not using FXVD4)
11- rpm -Uvh http://download.fedoraproject.org/pub/epel/6/i386/epel-release-6.5.noarch.rpm
12-yum install fpaste
13-yum install opencv-devel
14-yum install xawtv
15-chmod 664 /dev/video0
```
